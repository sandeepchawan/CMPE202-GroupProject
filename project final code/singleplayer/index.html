Compresso


About The Game

In Compresso, the player learns how compression can be done, here the player needs to identify the words that are repeated and drag the word and drop it into the compression box. Here the player needs to drag all the occurrences of the word.

In the game there are three levels, say easy medium and hard and with three different time intervals.



How To Play The Game

  • Select the mode and Time-Interval of the game.
  • Now, search for the words that has maximum frequency of occurrence, select the word and drag and drop the word into the compression box
  • Find all possible words such that maximum compression can be done
  • Once all the words are identified and dropped into the box click on Finish!!

Dont's

  • Do not select the word if is not repeated which results in score deduction
